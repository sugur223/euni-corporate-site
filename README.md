# 株式会社Euni コーポレートサイト

株式会社Euniのコーポレートサイト用WordPressカスタムテーマです。

## 企業理念

### Purpose（存在意義）

**人と人とのつながりが人生を拓く**

`Connect to possibilities.`

### Mission（使命）

```
豊かな発想と高い技術力によって、
善いつながりを育み、
すべての人々の可能性を広げる
```

### Vision（企業のありたい姿）

```
人と人との善いつながりを築く
インターフェースとして
信頼される存在になる
```

## 会社概要

- **会社名**: 株式会社Euni（ユニ）
- **英語表記**: Euni Inc.
- **事業内容**:
  - **つながりを創るIT事業** - 新しい出会いと関係性を生み出す（HP/LP制作、マッチングシステム、生成AI活用）

## クイックスタート

### 1. 必要な環境

- WordPress 6.0以上
- PHP 8.0以上
- MySQL 5.7以上 / MariaDB 10.3以上

### 2. インストール

```bash
# テーマフォルダをWordPressテーマディレクトリにコピー
cp -r euni-theme /path/to/wordpress/wp-content/themes/

# WordPress管理画面で有効化
# 外観 → テーマ → 「Euni Corporate Theme」を有効化
```

詳細なセットアップ手順は `docs/SETUP-LOCAL.md` を参照してください。

### 3. 初期設定

WordPress管理画面から「外観」→「カスタマイズ」→「会社情報」で以下を設定：

- 代表取締役名
- 代表者プロフィール写真・メッセージ・略歴
- 設立年月・資本金・所在地
- メールアドレス・電話番号・営業時間

## サイト構成

### トップページ

トップページは7つのセクションで構成されています：

1. **ヒーロー** - メインメッセージとタグライン
2. **ニュース** - 最新情報（最大5件）
3. **事業内容** - 3つの事業紹介
4. **企業理念（MVV）** - Purpose/Mission/Vision/Values
5. **代表メッセージ** - CEOからのメッセージ
6. **会社概要** - 基本情報テーブル
7. **お問い合わせ** - 連絡先情報＋フォーム

### 固定ページ

- **/philosophy/** - 社名に込めた想い（Eunoia/UNI/Uniの由来）

詳細な構成・コンテンツは `docs/HOMEPAGE_STRUCTURE_LATEST.md` を参照してください。

## 使用方法

### ニュースの投稿

1. WordPress管理画面から「ニュース」→「新規追加」
2. タイトルと本文を入力
3. カテゴリー選択（プレスリリース/お知らせ/イベント/メディア掲載）
4. 公開

トップページには最新5件が自動表示されます。

### お問い合わせフォーム

内蔵のフォーム機能を使用します。送信されたメールは設定したメールアドレスに届きます。

**フォーム項目**:
- お名前（必須）
- 会社名・組織名（任意）
- メールアドレス（必須）
- 電話番号（任意）
- お問い合わせ種別（必須）
- お問い合わせ内容（必須）

## カスタマイズ

### カラーの変更

`assets/css/euni-custom.css`のCSS変数を編集：

```css
:root {
    /* メインカラー */
    --color_main: #1a1a1a;
    --color_accent: #0066cc;
    --color_accent_hover: #0052a3;

    /* 背景カラー */
    --color-bg-light: #fafafa;
    --color-bg-white: #ffffff;
    --color-bg-gray: #f5f5f5;

    /* テキストカラー */
    --color-text-dark: #1a1a1a;
    --color-text-medium: #666666;
    --color-text-light: #999999;
}
```
