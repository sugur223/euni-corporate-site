### 3. 事業内容セクション (`#business`)

**目的**: IT事業を通じて「つながり」を創り、育てることを示す

**リード文**:
```
新しい出会いを生み出し、今ある関係を深める。
IT技術で「つながり」を創り、育てます。
```

**構造**: 2軸構成（IT事業メイン）
- つながりを創る（0→1）
- つながりを育てる（1→10）

**レイアウト**: 2カラムグリッド（デスクトップ）/ 縦積み（モバイル）

---

#### 3-1. つながりを創る

**コンセプト**: 新しい出会いと関係性を生み出す（0→1）

**説明文**:
```
新しい出会いと関係性を生み出します。
企業と顧客、人と人の最初の接点を創ります。
```

**代表的サービス**:
- HP/LP制作・Webアプリ開発
- マッチングシステム開発
- 生成AI活用・チャットボット開発

**詳細サービス例**（ドキュメント用）:
- コーポレートサイト・採用サイト・LP制作
- Webアプリケーション開発（予約システム、イベントプラットフォームなど）
- マッチングプラットフォーム開発（人材、メンター、ビジネスマッチングなど）
- 生成AI活用マッチング（AIによる最適な出会いの提案）
- AIチャットボット（問い合わせ対応、リード獲得）
- SEO・Webマーケティング

**CTA**: `お問い合わせ` → #contact

---

#### 3-2. つながりを育てる

**コンセプト**: 今ある関係を深め、継続させる（1→10）

**説明文**:
```
今ある関係を深め、継続させます。
顧客・社員との信頼関係を強化します。
```

**代表的サービス**:
- CRM・顧客管理システム開発
- コミュニティプラットフォーム開発
- データ分析・AI活用システム開発

**詳細サービス例**（ドキュメント用）:
- CRM/顧客管理システム開発
- マーケティングオートメーション連携
- コミュニティプラットフォーム開発（会員間交流機能、イベント管理など）
- 会員制サイト・ポータル開発
- 社内ポータル/イントラネット開発
- 社内SNS開発
- AIチャットボット（顧客サポート、社内FAQ）
- AIパーソナライゼーション（個別最適化されたコミュニケーション）
- エンゲージメント分析・BI構築
- A/Bテスト・改善支援

**CTA**: `お問い合わせ` → #contact

## 事業内容定義

### 定義フォーマット

各事業は以下の要素で構成されています：

```
【誰に】（ターゲット）
【どんな困りごとを】（課題）
【どう解決し】（提供価値）
【どんな変化をもたらすか】（成果）
```

---

### 1. 成長支援コミュニティ事業

| 要素 | 内容 |
|------|------|
| **誰に** | キャリアに悩みを持つ個人（年齢・職種問わず） |
| **困りごと** | 次のキャリアが見えない／相談できる相手がいない／学びの機会が限られている |
| **解決方法** | 学びと交流を両立したコミュニティの場を提供し、多様な価値観に触れる機会を創出 |
| **もたらす変化** | 新たな視点や選択肢に気づき、自分らしいキャリアの一歩を踏み出せる |
| **Mission体現** | 「善いつながり」→学びと対話の場／「可能性を広げる」→自分らしい選択 |

---

### 2. ITソリューション事業

| 要素 | 内容 |
|------|------|
| **誰に** | コミュニティ運営・組織内のつながりづくりに取り組む企業・団体 |
| **困りごと** | プラットフォーム構築のノウハウがない／エンゲージメント維持が難しい／ITリソース不足 |
| **解決方法** | コミュニティプラットフォームの企画・開発から運用支援、AI活用まで一貫サポート |
| **もたらす変化** | メンバー同士が自発的につながり、継続的な関係性が生まれる仕組みを実現 |
| **Mission体現** | 「高い技術力」→システム開発・AI活用／「善いつながり」→自発的なつながり促進 |

---

### 3. コンサルティング事業

| 要素 | 内容 |
|------|------|
| **誰に** | 組織内のつながりや関係性を強化したい企業 |
| **困りごと** | 心理的安全性が低い／組織の一体感が薄い／人材育成の仕組みが機能していない |
| **解決方法** | 組織開発・人材育成プログラムの設計と実行支援で、関係性の質を高める |
| **もたらす変化** | 一人ひとりが安心して力を発揮でき、持続的に成長し続ける組織文化が根づく |
| **Mission体現** | 「善いつながり」→組織内のつながり強化／「可能性を広げる」→安心して力を発揮 |

## デザインシステム

### カラースキーム

```css
/* メインカラー */
--color-text-primary: #1a1a1a;
--color-accent: #0066cc;

/* 背景カラー */
--color-bg-white: #ffffff;
--color-bg-light: #fafafa;
--color-bg-gray: #f5f5f5;

/* グレースケール */
--color-gray-100: #f5f5f5;
--color-gray-300: #e0e0e0;
--color-gray-500: #9e9e9e;
--color-gray-700: #616161;
```

**使用原則**:
- すべてのセクションで統一されたカラーパレットを使用
- グラデーションは装飾要素にのみ使用（背景には使用しない）
- 白・グレー・黒を基調としたシンプル&ミニマルなデザイン
