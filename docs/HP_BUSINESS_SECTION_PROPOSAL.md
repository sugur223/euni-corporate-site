# HP事業内容セクション - 掲載案

**作成日**: 2025年11月5日
**目的**: HP用に絞り込んだ事業内容の表示案を作成

---

## 基本方針

### 掲載する情報の粒度
- **詳細すぎない**：すべてのサービスを列挙しない
- **わかりやすい**：顧客が自分に必要なものを即座に判断できる
- **技術力を示す**：IT企業としての専門性を伝える
- **「つながり」軸を明確に**：MVVとの一貫性

---

## 提案：2軸 × 代表的サービスのみ掲載

### セクション全体構成

```
┌─────────────────────────────────────────────┐
│           事業内容（BUSINESS）              │
├─────────────────────────────────────────────┤
│ セクションラベル: BUSINESS                  │
│ 見出し: 事業内容                             │
│ リード文: IT技術で「つながり」を創り、育てる │
└─────────────────────────────────────────────┘

┌─────────────────┬───────────────────────────┐
│ つながりを創る   │  つながりを育てる         │
├─────────────────┼───────────────────────────┤
│ （カード1）     │  （カード2）              │
└─────────────────┴───────────────────────────┘
```

## 案1：シンプル版（代表的サービス3つずつ）

### つながりを創る

**アイコン**: 人と人が出会うイメージ（現在のコミュニティアイコンを流用可）

**見出し**: つながりを創る

**説明文** (60-80文字):
```
新しい出会いと関係性を生み出すIT技術を提供します。
Webサイト制作、マッチングシステム、AIチャットボットで、
企業と顧客、人と人の最初の接点を創ります。
```

**代表的サービス** (箇条書き):
- **Webサイト・アプリ開発** - 企業と顧客をつなぐ
- **マッチングプラットフォーム** - 最適な出会いを創出
- **AIチャットボット** - 24時間対応の初期接点

**CTA**: `お問い合わせ` → #contact

---

### つながりを育てる

**アイコン**: つながりが深まるイメージ（現在のコンサルティングアイコンを流用可）

**見出し**: つながりを育てる

**説明文** (60-80文字):
```
既存の関係性を深め、継続させるIT技術を提供します。
CRM、コミュニティプラットフォーム、データ分析で、
顧客・社員との信頼関係を強化します。
```

**代表的サービス** (箇条書き):
- **CRM・顧客管理システム** - 関係性を管理・深化
- **コミュニティプラットフォーム** - メンバー間の交流促進
- **データ分析・AI活用** - 関係性の質を可視化・改善

**CTA**: `お問い合わせ` → #contact

## 推奨：案1（シンプル版）

**理由**:
1. **適度な情報量** - 詳しすぎず、簡潔すぎない
2. **具体性がある** - 代表的サービスで何ができるか伝わる
3. **デザインしやすい** - 現在の3カード構成から2カードへの変更が容易
4. **モバイル対応** - 情報量が適切で読みやすい

## サブ事業（コンサルティング）の扱い

### 表示方法の選択肢

**選択肢A**: 事業内容セクションに小さく追記
```
┌─────────────────────────────────────────────┐
│ その他：コンサルティング・運営支援          │
├─────────────────────────────────────────────┤
│ 成長支援コミュニティ運営、組織開発コンサル │
└─────────────────────────────────────────────┘
```

**選択肢B**: 別セクション化（簡易版）
```
┌─────────────────────────────────────────────┐
│        コンサルティング・運営支援           │
├─────────────────────────────────────────────┤
│ IT導入だけでなく、運営支援や組織開発も     │
│ サポートします。                            │
└─────────────────────────────────────────────┘
```

**選択肢C**: 記載しない
- IT事業のみに集中
- 問い合わせ時に対応

**推奨**: 選択肢A（小さく追記）
- IT事業がメインであることを明確にしつつ
- サブ事業も提供していることを伝える

## 現行デザインからの変更点

### 変更前（現行）
- 3つのカードを縦に配置
- 各カード：見出し + 説明文 + CTA

### 変更後（提案）
- 2つのカード（＋サブ事業小）
- デスクトップ：2カラム（左右配置）
- モバイル：縦積み
- 各カード：アイコン + 見出し + 説明文 + サービス箇条書き + CTA

### CSS調整必要箇所
- `.p-businessStack` → 2カラムレイアウトに変更
- サブ事業用の新しいスタイル追加（小さめのカード）
