# トップページ改善に向けたMVV整理

## 1. ドキュメントの目的
- 現状のトップページ（`front-page.php`）で表現されているメッセージを整理し、ミッション・ビジョン・バリュー（MVV）が所定の定義を満たせているか確認する。
- MVVを軸に、トップページで強化すべき情報やストーリー構成の方向性を明らかにする。

## 2. トップページの主要メッセージ構成
- **ヒーローコピー**：「つながろう あなたの未来のために」「一つの出会いが、新しい道を開く。」
- **パーパス（Purpose）**：「人と人をつなぎ 未来をつくる」および `Connect to possibilities.` のタグライン。
- **MVVセクション**
  - Vision（目指す未来）：誰もがつながりの中で生きる社会
  - Mission（使命）：人と人をつなぐ接点となり、善いつながりを育む。
  - Values（私たちの価値観）：
    1. 問い続けよう — 本当に大切なことは何か。当たり前を疑って、本質を探ろう。
    2. 心から楽しもう — 人生は一度きり。遊び心を持って、わくわくしながら、やってみよう。
    3. 循環を創ろう — 受け取ったものを、また誰かに。善意が巡る流れを創ろう。
    4. 矛盾を受け入れよう — 完璧じゃなくていい。矛盾も、不完全も受け入れる。それでも、少しずつ前へ。
- **リンクセクション**：企業理念（社名の由来）と事業概要（3事業）への導線。

## 3. 定義（すり合わせ）
- **Mission（ミッション）**：会社はどのような社会を目指し、どのような価値を提供するのか。
- **Vision（ビジョン）**：企業が「あるべき・ありたい姿」を言葉にしたもの。リーダーが描く目標像そのものであり、メンバーの物事の見方や考え方を方向付ける基本的な枠組となる。
- **Value（バリュー）**：ミッションとビジョンを実現するための具体的な行動様式や判断基準。

## 4. 現状評価とギャップ
- **Vision**
  - 「誰もがつながりの中で生きる社会」と理想状態を明確に示せている。
  - 理想の社会像の具体性（時間軸・対象領域など）がやや抽象的で、ヒーローコピーとの関連が伝わりにくい。
- **Mission**
  - 「人と人をつなぐ接点」「善いつながりを育む」という価値提供の方向性は示されている。
  - 社会に対するインパクト（どの領域で、誰に対し、どんな変化を起こすか）がもう一段語れると、「どのような価値を提供するのか」がより明瞭になる。
- **Values**
  - 全体として内面の姿勢を提示しており、MVVの一貫性がある。
  - 具体的な行動シーン（例：プロダクトづくり、顧客との接点、採用）との紐付けが弱く、「どう判断するか／動くか」の具体度が不足。
- **ストーリー全体**
  - 表示順が Vision → Mission になっており、MVVの定義（Mission → Vision → Values）とズレがあるため、読み手が使命と理想像の違いを理解しにくい。
  - Purpose → Vision → Mission → Values の流れは成立しているが、各ブロック間に「なぜその結論に至るのか」を補完する説明が少なく、読み手視点で世界観が段階的に深まる構成になっていない。
  - 事業紹介との接続が弱く、MVVが実際のサービスや実績にどのようにつながるかの橋渡しが必要。

## 5. MVV再設計（提案）
- **Mission（使命）候補**
  - **推奨**：`つながりの場をつくり、善意が巡る社会を広げる。`
  - **代替**：`人と人の接点を整え、挑戦する人の味方になる。`
  - 行動（場をつくる／接点を整える）と目的（善意が巡る／挑戦を支える）を一文で言い切れる形に統一。
- **Vision（ビジョン）候補（企業のありたい姿）**
  - **推奨**：`Euniは、人と人を信頼で結ぶハブとして頼られている。`
  - **代替**：`Euniは、挑戦者のそばで機会をひらく伴走チームとして選ばれている。`
  - Missionが実現した先に、自社がどんな立場・評価で語られるかを端的に示す。
  - **拡張表現（参考コピー）**
    - `Euniは、人と人が安心して集まれる信頼のハブとして、善意の循環を生み出すプラットフォームになっている。`
    - `Euniは、オンラインとリアルをつなぐ接点づくりの専門チームとして選ばれ、誰もが必要なつながりにアクセスできる環境を広げている。`
    - `Euniは、地域や業界を超えて挑戦者に伴走し、つながりで社会課題を解決するエコシステムの中心で新しいモデルを生み出し続けている。`
- **Vision選定の観点**
  - Missionで掲げた価値提供の先に、自社がどう評価・認知され、どんな役割を担っているかを示せているか。
  - 社員が日常の意思決定で「この方向に向かっているか？」を判断する羅針盤になるか。
  - リーダーシップメッセージとして、ステークホルダーに未来像を想起させ、共感や行動を促せる構造になっているか。
- **言葉選び・一貫性のガイド**
  - わかりやすい言葉：専門用語を避け、「味方」「伴走」など誰にでもイメージできる言葉を選ぶ。
  - 情報量は最小限：一文で行動と目的を示し、詳細はサブテキスト（詳細資料や各セクション）で補う。
  - 社会性の明示：Missionで社会にもたらす価値を語り、Visionで自社がどのような姿勢・能力で社会に影響を与えているかを描写する。
  - MVVの一貫性：Missionで示した価値提供（場づくり・接点づくり・伴走）をVisionとValuesにも反映させ、キーワードやトーンを揃える。
- **Values（価値観・行動指針）再整理**
  1. **問い続けよう** — 「プロジェクト開始時に、解決したい本質課題を言語化し、仮説検証を繰り返す。」
  2. **信じ抜こう** — 「人は誰でも過ちを犯す。目の前の言動だけで決めつけず、その人の背景まで想像して向き合い続ける。」
  3. **心から楽しもう** — 「チーム内の挑戦や小さな成功を共有し、前向きな雰囲気をつくる。」
  4. **循環を創ろう** — 「得た知見・リソースを外部にも還元し、共創機会を広げる。」
  5. **矛盾を受け入れよう** — 「完全解を求めず、プロトタイプと対話を重ねて最適解を探す。」
  - 既存の価値観表現を活かしつつ、行動例を付記して判断基準として使いやすくする。
  - Missionとのつながり：①問い続けよう → 的確な場づくり、②信じ抜こう → 困難な状況でも味方でいる、③心から楽しもう → 継続可能な接点運営、④循環を創ろう → 善意が巡る社会づくり、⑤矛盾を受け入れよう → 伴走しながら課題を解くスタンス。
- **表示順と導線**
  - トップページでは Mission → Vision → Values の順に再配置し、定義の流れがそのまま読み手に伝わるようにする。
  - Missionの直後に Vision への橋渡し文（「その結果、私たちが目指す姿は…」など）を挿入し、ストーリーが自然につながる構成にする。

### Hero & Purpose刷新案（確定版：2025年11月）
- **Heroコピー（`p-hero__subtitle / title / lead`）**
  - Subtitle：`Connect to possibilities.`
  - Title：`つながりで、<br>未来をひらく。`
  - Lead：`一人ひとりの可能性が、誰かとつながることで広がっていく。`
  - ボタン文言：`もっと知る ↓`
  - 狙い：MissionをそのままHero Titleに配置。最初に企業の使命を明確に伝え、Purposeタグラインとも統一。シンプルで力強いメッセージ。

- **Purposeステートメント（`p-mvvPurpose`）**
  - 見出し：`出会いが、人生を変える瞬間がある。`
  - 本文：
    ```
    ふとした会話が、新しい道を見せてくれた日。
    誰かの言葉が、背中を押してくれた日。

    Euniは、そんな出会いと関係が生まれる場を整えます。
    一人ひとりの可能性が、誰かとつながることで広がっていく。
    迷いながらでも、一歩ずつ前へ進める関係を、もっと当たり前にしたい。

    人と人がつながることで、未来は変わっていく。
    ```
  - タグライン：`Connect to possibilities.`
  - 狙い：具体的な情景（ふとした会話、背中を押してくれた日）から始め、企業姿勢は控えめに。読者が自分の体験と重ねられる明るく希望的な表現。

### MVV確定版（2025年11月）
- **Mission（使命）**
  - `つながりで、未来をひらく。`
  - 特徴：シンプルで力強い。「つながり」（手段）→「未来をひらく」（結果）が明快。Heroの「新しい一歩を踏み出せる。未来は、ここから始まる。」と呼応。

- **Vision（企業のありたい姿）**
  - `Euniは、善いつながりを築くインターフェースとして信頼されている。`
  - 特徴：社名の由来（UNI = User Network Interface）と直結。「善いつながり」で質を明示し、「信頼されている」で「信じ抜こう」のValueと連動。企業の役割とポジションを明確に示す。

- **Values（私たちの価値観）**
  1. **問い続けよう** — 本当に大切なことは何か。当たり前を疑って、本質を探ろう。
  2. **人として信じよう** — 誰もが弱さを抱えている。見えている姿だけで判断せず、人として信じ合う。
  3. **心から楽しもう** — 楽しむことを、忘れない。遊び心を持って、わくわくしながら、やってみよう。
  4. **循環を創ろう** — 受け取ったものを、また誰かに。善意が巡る流れを創ろう。
  5. **矛盾を受け入れよう** — 完璧じゃなくていい。矛盾も、不完全も受け入れる。それでも、少しずつ前へ。

### 感情の核と語り口
- **大切にしたい感情**
  - 出会いや関係性が自分を支えてくれたという深い感謝。
  - 苦しみの渦中にいる人へ寄り添い、前を向くきっかけを渡したいという強い意志。
  - 喜びと痛みが共存するリアルな人生経験を肯定するまなざし。
- **言葉に落とし込むポイント**
  - “誰かの支えで今がある” という感謝を〈企業としての姿勢〉に変換し、「私たちも支え手になる」という約束にする。
  - 苦しみに寄り添う姿勢を、Missionでは「味方になる」、Visionでは「頼られる存在」という言葉で表現。
  - 自分語りにならないよう、「あなた」「私たち」という主語をバランスよく使い、読者の体験に余白を残す。
- **コピーの方向性（例）**
  - ヒーローリード案：`一人では越えられない夜がある。だから、つながりの灯りを絶やさない。`
  - Mission補足文：`苦しみの底にいる誰かが、もう一度前を向けるように。私たちはつながりの場を整え続ける。`
  - セクション間の橋渡し：`出会いが人生を変えると信じるからこそ、Euniはいつも誰かのそばで機会をひらく伴走者でありたい。`
  - Values導入：`感謝を循環させ、同じように誰かの力になれるチームであり続けるための約束。`

## 6. 改善アイデア（方針案）
- **ストーリーテリングの強化**
  - ヒーローセクションに「Missionに紐づく社会課題」や「Visionにつながる未来像」の一文を追加し、読み手がすぐに世界観を理解できるようにする。
  - Purpose の段落に Mission のキーワード（接点・善いつながり等）を適切に混ぜ、メッセージの一貫性を高める。
- **Vision / Mission の具体化**
  - Vision の文章に「どの領域・人々がどう変わった状態か」を加筆し、理想の絵姿を描写する。
  - Mission を「どんなサービス・アクションで価値を提供するか」の観点で補強し、事業概要への導線を強める。
- **Values の実務連動**
  - 各バリューの下に「具体的な姿勢・判断例（例：プロジェクト進行時／ユーザー対応時）」を短文で追加し、採用や共創の相手にも伝わりやすくする。
  - バリューを支える実話やエピソードへのリンクを用意すると信頼性が上がる。
- **MVV と事業紹介の接続**
  - 事業概要セクション冒頭に「Mission を体現する3つの取り組み」といった概念整理を入れる。
  - 各事業カードに「該当するバリュー」「届けたい価値」をアイコンや短いフレーズで表示し、MVVとのひもづけを可視化する。
- **証拠・社会的証明の追加**
  - Mission/Value を裏付ける実績や数字、顧客の声などを差し込むことで、メッセージが抽象語に留まらないようにする。

## 8. トップページ全体の改善案
- **想定する構成フロー**
  1. Hero（課題共有＋共感）
  2. Mission（何を・なぜ）
  3. Vision（どんな姿になる）
  4. Values（どう行動する）
  5. Story/Origin（社名の由来・背景）
  6. Business（具体的な取り組み）
  7. Voices/Proof（実績・パートナーの声）
  8. CTA（問い合わせ／資料／イベント案内）
- **Heroセクション（`front-page.php` ヒーローブロック）**
  - 追加コピー案を採用し、つながりが支えになる理由を一言で提示。
  - スクロール誘導ボタンを追加し、次セクション（Mission）へスムーズに案内。
- **Missionセクション（現MVVブロック冒頭）**
  - Mission文を案1/案2のどちらかに確定し、Visionと順序を入れ替え。
  - Missionの下に「今向き合っている社会課題」を1行で補足（例：「孤立や選択肢の不安を抱える人に伴走します」）。
- **Visionセクション**
  - Vision案を確定し、Missionの直後に配置。
  - Visionの下に、達成イメージを象徴する写真またはグラフィックを1点置き、読後のイメージを喚起。
- **Valuesセクション**
  - 「問い続けよう／信じ抜こう／心から楽しもう／循環を創ろう／矛盾を受け入れよう」の5項目へ差し替え。各項目ごとに象徴アイコンと行動例を1行で表示。
  - セクション最後に「Valueを体現したエピソードへのリンク」（ブログ／note等）が取れる場合は設置。
- **Origin（社名の由来）**
  - 現行の長文を整理し、要約＋詳細（アコーディオン）形式へ。最初に「感謝とつながりへの敬意」という共通テーマを明示。
  - 感情の核（感謝・寄り添い）を短い引用ボックスで表示し、Heroとメッセージを往復させる。
- **Businessセクション**
  - 各事業カードに「Missionとの接点」「支えている人」を1フレーズずつ追記。
  - CTAボタン（資料請求／相談会予約など）を事業セクション直下に配置し、具体的な行動につなげる。
- **Voices / Evidence（新規セクション案）**
  - 顧客の声、共創パートナーコメント、定量実績などを最大3件掲載。Mission・Visionの信頼性を補強。
  - QuotesはValue「信じ抜こう」「循環を創ろう」に共鳴する内容を選ぶ。
- **最後のCTA**
  - 「まずは話してみませんか？」など共感ベースのCTAに変更。
  - お問い合わせ・資料ダウンロード・イベント/コミュニティ案内の3導線があればボタンで並列配置。
- **デザイン/UXチェック**
  - パララックス装飾の数を精査し、読みやすさとアクセシビリティを優先。
  - モバイル表示でのフォントサイズ・行間・余白を見直し、HeroとCTAがファーストビューで読み取れるようにする。
  - alt属性やARIAラベルを整備し、視覚的な演出に頼らない情報設計を行う。
- **運用／検証の提案**
  - 主要セクションをCMS化（ACFなど）し、コピーのABテストや実績の更新を容易にする。
  - 公開後、問い合わせ数・滞在時間・スクロール率などを計測し、MVV訴求が行動につながっているかを確認する。

## 7. 次のアクション案
- Mission / Vision の改稿案を複数パターン用意し、経営層レビューで最終文言を確定する。
- Values の具体行動例について、チームメンバーへのヒアリングや既存事例の棚卸しを実施する。
- `front-page.php` のMVVセクションを Mission → Vision → Values の順に再設計し、橋渡しコピーと行動例の掲載場所を決めるワイヤーフレームを作成する。
- MVVコピー確定後、HeroやPurposeセクションのテキストも整合性チェックを行い、必要に応じて修正案を起こす。
